# Paper

A wallpaper daemon for Wayland compositors implementing the layer-shell protocol.

## Features

- Supports png and jpg format
- Tiled wallpapers
- Bordered wallpapers

## Dependencies

- [snui](https://gitlab.com/snakedye/snui.git)

## Building

```shell
git clone https://gitlab.com/snakedye/paper.git
cd paper
cargo build --release
```
